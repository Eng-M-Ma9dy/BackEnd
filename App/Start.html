    * How Tow Start New Project     
    ----------------------------------------------
    
    For Start New Project
    
    01. Install Node.JS From Here
    
    02. Create New Folder Included index.js File
    
    03. Open CMD in VS Code ( Ctrl + ` )
    
    04. Set Command >> npm init -y
    05. Set Command >> npm i express
    06. Set Command >> npm i mysql2             |  In Case Use SQL   DB
    07. Set Command >> npm i sequelize          |  In Case Use SQL   DB
    08. Set Command >> npm i mongoose           |  In Case Use Mongo DB

    09. Set Command >> npm i dotenv             |  For Set Secret Info
    10. Set Command >> npm i joi                |  For Do Validation
    11. Set Command >> npm i jsonwebtoken       |  For Create Token
    12. Set Command >> npm i bcrypt             |  For Hash Password
    13. Set Command >> npm i nodemailer         |  For Send Email
    14. Set Command >> npm i slugify            |  For Rename Product Name
    15. Set Command >> npm i cors               |  For Link Front With Back By Different Port 
    16. Set Command >> npm i morgan             |  For Get Info About Api Call
    17. Set Command >> npm i qrcode             |  For Make QR Code
    18. Set Command >> npm i multer             |  For Upload File
    19. Set Command >> npm i nanoid             |  For Rename File User Upload It
    20. Set Command >> npm i cloudinary         |  For Upload File to Cloud

    21. Set Command >> npm install -g nodemon   | First use nodemon Only
    
    22. Set The Type = Module in package.json {"type":"module",}
    
    23. For Run App Server >> nodemon index.js
